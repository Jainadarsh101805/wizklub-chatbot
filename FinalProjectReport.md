## 1. Executive Summary
This report details the design, development, and implementation of a next-generation AI Chatbot for **WizKlub**. The objective was to create a high-conversion lead generation tool that segments audiences (Parents vs. Schools) while providing a premium, "Innovation Lab" digital experience.

## 2. Project Definition
The WizKlub AI Chatbot is an interactive web-based interface designed to:
- **Identity Segment**: Distinguish between retail (Parents) and institutional (Schools) visitors.
- **Lead Capture**: Collect valid Name, Email, and Phone number in a zero-friction conversational flow.
- **Value Delivery**: Assess the child's "STEM Quotient" or school's needs before final conversion.
- **Branding**: Reinforce WizKlub's position as a premium technology educator through state-of-the-art UI/UX.

## 3. Technical Stack
- **Frontend**: React.js
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Celebrations**: Canvas-Confetti
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom Glassmorphic System)
